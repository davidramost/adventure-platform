## 📐 Reglas Globales del Workspace

### Seguridad
- **CORS** configurado para dev y producción
- **JWT** en headers `Authorization: Bearer {token}`. Nunca en URLs.
- **Passwords** siempre hash con BCrypt (strength 12)
- **Credenciales** en variables de entorno. Nunca hardcoded.
- **HTTPS** en producción

### Arquitectura
```
Usuario (React 19)
    ↓ HTTPS
Spring Boot (REST API + JWT)
    ↓ JDBC
TiDB Cloud (MySQL compatible)
```

### Git
- Commits descriptivos en español o inglés
- No subir `.env` ni credenciales
- `.gitignore`: `node_modules/`, `target/`, `.env`, `dist/`

## 🚀 Quick Reference

### Frontend (tfg_adventure)
```bash
cd tfg_adventure
npm run dev          # Desarrollo local (http://localhost:5173)
npm run build        # Build producción → dist/
```

### Backend (tfg_backend)
```bash
cd tfg_backend
./mvnw spring-boot:run                    # Desarrollo local
./mvnw clean package -DskipTests          # Build JAR
docker build -t tfg-backend .             # Docker image
```
